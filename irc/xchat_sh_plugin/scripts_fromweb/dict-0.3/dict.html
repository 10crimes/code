Dict.py

XChat Scipt by Jens Taylor

About

This script allows XChat to lookup word's definitions via http://dictionary.com. If successful, the definition is printed for everyone in the channel to see. Red error messages are printed if anything does not work, indicating the problem.

Download and Installation

Download the lasted release, 0.3. Download now!

To install, simply extract the file to any directory. If extracted to your ".xchat2/" directory, the script will load when XChat starts. Otherwise, you'll have to start it yourself.

tar -xjf dict.tar.bz2
Optionally, also move it to your ".xchat2/" directory.
mv dict.py ~/.xchat2/

Usage

First, the script needs to be loaded into XChat. This can be done in two ways: you can either restart XChat (with the dict.py file in the ".xchat2/" directory), or you can load it manually.

In any XChat tab, simply type the following command
/load path/to/dict.py
If you want to unload this module, simply type
/unload dict.py

Note: "path/to/" is only necessary if the file is not in either your home directory or the ".xchat2/" directory.

Once loaded, the script can be used by typing "!define keyword". An example follows.

<jens> !define moo
<jens> -moo-
<jens> n : the sound made by a cow or bull v : make a low noise, characteristic of bovines

Notes and Known Issues

  • There is a sixty second timeout between each query to avoid abuse. It can easily be changed with the "/dict time <x>" command, where "x" is a time in seconds.
  • *FIXED* XChat hangs for ~1-3 seconds while executing this script. This is because XChat does not continue to work while a script is running. The reason it takes this long is because the connection to download the source may take several seconds to establish. This issue will be worked out in the next release.
  • *FIXED* When you use the script and query for a definition, the definition is printed to the channel before your original query, making it look very awkward. This will also be fixed in the next release.
  • This script was written and tested with XChat 2.4.1. With some tweaking, it could be changed to work with any program.
