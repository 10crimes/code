From ExoticA

Jump to: navigation, search

This is an interface to the Modland music collection 3 (Full Instructions). The search uses keywords, phrases and boolean operators (+ - and | before a word). By default the search will match results that contain all entered words. +apples -oranges will return matches that contain apples but not oranges. For a phrase wrap it in quotes "apples and oranges". |apples |oranges will return matches that contain either apples or oranges. Use the checkboxes to do an exact match or search for non word strings. The quick search matches against Filename, Author, Format and Collection fields.
